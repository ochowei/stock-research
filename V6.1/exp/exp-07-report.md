# EXP-V6.1-07: Next-Day Suitability Classifier (AI Filter)

**實驗日期:** 2025-12-24
**核心目標:** 在已知「跳空 (Gap > 0.5%)」的前提下，利用機器學習 (XGBoost) 預測當日交易是否值得執行。

## 1. 實驗結果 (Results)

實驗數據表明，加入 AI 濾網後，策略績效發生了質的飛躍。

| Metric | Baseline (無腦做) | Model (AI 過濾) | 差異 (Diff) |
| :--- | :--- | :--- | :--- |
| **Trade Count** | 14,543 | 7,521 | -48.3% (過濾掉一半雜訊) |
| **Win Rate** | 51.12% | **59.97%** | **+8.85%** (關鍵突破) |
| **Avg Return** | 0.046% | **0.978%** | **+0.931%** (由損益兩平變暴利) |

* **解讀：** Baseline 的平均獲利 (0.046%) 扣除滑價後幾乎無利可圖。而 Model 選出的交易平均獲利接近 1%，即使扣除成本仍有巨大優勢。

## 2. 特徵重要性 (Feature Importance)

模型「學到了什麼」？

1.  **RSI_14 (47.9%):** **絕對核心。** 模型學會了識別「均值回歸」的機會。若跳空但 RSI 不夠高（未過熱），則容易變成趨勢盤 (Gap and Go)，不適合反向操作。
2.  **VIX (17.8%):** **環境感知。** 確認了恐慌指數對 Gap 策略的勝率有決定性影響。
3.  **Vol_Ratio (11.9%):** **籌碼結構。** 昨日爆量通常意味著情緒耗盡，更有利於反轉。

## 3. 結論 (Conclusion)

* **驗證成功 (Hypothesis Confirmed):** 我們成功將「選股問題」轉化為「擇時問題」。
* **執行方案:** 將 Exp-07 模型部署至實盤，作為 Gap > 0.5% 之後的第二道濾網。
